# PyDesigner

PyDesigner is a hands-free diffusion MRI preprocessing and modeling pipeline for DTI, DKI, FBI, FBWM, and related diffusion MRI analyses.

## Version

Current release: `2.0.0`

## Requirements

PyDesigner 2.0.0 requires:

- Python `>=3.14`
- MRtrix3 available on your `PATH`
- FSL available on your `PATH`
- uv for Python environment and package installation

The PyDesigner wheel is a pure-Python universal wheel:

```text
pydesigner_dwi-2.0.0-py3-none-any.whl
